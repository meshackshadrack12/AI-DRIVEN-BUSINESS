# Smart Ranch Dashboard

An AI-powered web application for livestock monitoring, built exclusively with HTML, CSS, and Vanilla JavaScript.

## Features
- **AI-Powered Livestock Monitoring Simulation**: Dummy data and movement algorithms.
- **GPS Map Integration**: Uses Leaflet.js for real-time tracking visualization.
- **Dynamic Alerts**: Health status and abnormal movement indicators.
- **Modern Light Yellow Theme**: High-contrast, premium, mobile-responsive UI using primary colors.
- **Service Tiers**: Outlined service offerings for agricultural businesses.
- **Functional Contact Page**: Reach out for support or inquiries.

## Setup Instructions
1. Navigate to the project directory.
2. Open `index.html` in your preferred web browser.
3. **Note:** For best results and to ensure all JavaScript (especially if using modules) runs correctly without CORS issues, serve the files using a local HTTP server (e.g., VS Code Live Server, or Python's `python -m http.server`).

## Architecture
- `index.html` - Landing page, Service Tiers, Login
- `contact.html` - Contact Form
- `dashboard.html` - Main tracking application
- `css/styles.css` - Global and component styling
- `js/` - Logic for auth, data simulation, mapping, and dashboard UI
